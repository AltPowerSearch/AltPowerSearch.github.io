Programatic Search Access

Alt-Power Search can be configured to return search results proactively (at initiation), by setting the ?q= search query parameter when constructing the URL. You can add this search engine to your web browser, such as Firefox (simply right-click on the location bar) or Google Chrome (and thus Microsoft Edge, Brave, Opera & Vivaldi), by setting the search URL field to /?q=%s .
